# グラデーションマップ for YMM4

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](#)
[![.NET](https://img.shields.io/badge/.NET-10.0-purple.svg)](#)
[![Release](https://img.shields.io/github/v/release/routersys/YMM4-GradientMap.svg)](https://github.com/routersys/YMM4-GradientMap/releases)

---

YukkuriMovieMaker4（YMM4）上で動作する、**グラデーションマップ映像エフェクトプラグイン**です。
映像の輝度情報をグラデーションカラーに置き換えることで、色調補正・カラーグレーディング・アーティスティックな色変換を実現します。
Photoshop 互換の `.grd` ファイル読み込み、画像ファイルからのグラデーション生成、インラインのグラデーションエディタによるリアルタイム編集、HLSL ピクセルシェーダーによる GPU アクセラレーション描画、27 種類の合成モードを搭載しており、プロフェッショナルな色彩表現を YMM4 上で強力にサポートします。

## 概要

本プラグインは YMM4 の「映像エフェクト」として動作するグラデーションマップエフェクトです。
映像のピクセルごとに輝度（`0.299R + 0.587G + 0.114B`）を計算し、その値をキーとしてグラデーションテクスチャからカラーをルックアップすることで、元映像の明暗をベースにした色変換を行います。
グラデーションの定義方法は、Photoshop `.grd` ファイルの読み込み、各種画像ファイル（PNG / JPG / BMP / GIF / TIFF）からのサンプリング、プロパティパネル内蔵のインラインエディタによる直接編集の 3 通りに対応しています。

## インストール方法

1. [Releases](https://github.com/routersys/YMM4-GradientMap/releases) ページから最新のプラグインファイル（`.ymme`）をダウンロードしてください。
2. YMM4 が起動していないことを確認し、ダウンロードしたファイルを実行（ダブルクリック）してインストールします。
3. YMM4 を起動し、タイムライン上のアイテムを選択します。
4. 映像エフェクトの追加から **「グラデーションマップ」** を選択してください。

## 主な機能

### 1. GPU アクセラレーション描画

各ピクセルの輝度計算、グラデーションテクスチャのサンプリング、27 種類の合成モードの演算、不透明度によるブレンディングがすべて GPU 上で完結するため、CPU を占有せず、高フレームレートでのリアルタイムプレビューが可能です。

### 2. Photoshop GRD ファイル完全対応

Adobe Photoshop のグラデーションプリセットファイル（`.grd`）をバイナリレベルで解析するパーサーを独自実装しています。

- **対応バージョン**: レガシーフォーマット（v3/v4）および v5 ディスクリプタ形式
- **対応カラーモデル**: RGB（RGBC）、HSB（HSBC）、CMYK（CMYC）、Lab（LbCl）、グレースケール（Grsc）
- **マルチグラデーション**: 1 つの `.grd` ファイルに含まれる複数グラデーションのマニフェスト読み取りと、インデックスによる個別選択に対応
- **カラーストップ/透明ストップ**: 位置とミッドポイントを考慮した高精度な補間サンプリング

### 3. インライングラデーションエディタ

プロパティパネルに直接組み込まれた WPF ベースのグラデーションエディタです。外部ファイルを用意しなくても、その場でグラデーションを作成・編集できます。

- **カラーストップの追加**: グラデーションバー上をクリックすることで、任意の位置にカラーストップを追加
- **ドラッグ操作**: カラーストップマーカーをドラッグして位置を自由に調整
- **カラーストップの削除**: コンテキストメニューから不要なカラーストップを削除
- **リアルタイムプレビュー**: カラーストップの変更が即座に `LinearGradientBrush` に反映され、プロパティパネル上でグラデーションの見た目を確認可能
- **JSON シリアライズ**: エディタの状態は JSON 形式でプロジェクトファイルに保存され、完全に再現可能

### 4. 27 種類の合成モード

Photoshop を含む主要な画像編集ソフトウェアと同等の 27 種類の合成モードを、すべて HLSL シェーダー内で実装しています。グラデーションマップで置き換えた色と元映像の色を、選択したモードで合成することで、多彩な表現が可能です。

### 5. サムネイル付きカスタムファイルセレクター

WPF（`CustomFileSelector.xaml`）で実装された独自のファイル選択 UI を搭載しています。

- **ディレクトリ連動**: 選択したファイルと同じフォルダにある対応ファイル（`.grd`, `.png`, `.jpg`, `.jpeg`, `.bmp`, `.gif`, `.tiff`）を自動的にリストアップ
- **非同期サムネイル生成**: リスト内のファイルをバックグラウンドで非同期にレンダリングし、グラデーションプレビューをサムネイルとして表示
- **`.grd` ファイルのツールチップ**: `.grd` ファイルには「Photoshop 対応ファイル」の識別ツールチップが表示される特別な UI 処理を適用

### 6. GRD インデックスセレクター

`.grd` ファイル内に複数のグラデーションが含まれている場合、サムネイルと名前を表示するドロップダウン UI（`GrdIndexSelector.xaml`）が自動的に表示されます。
マニフェスト情報（各グラデーションの名前・インデックス）は `GrdManifestReader` によりキャッシュ付きで読み取られ、同じファイルを再度選択した際に再パースが発生しません。

### 7. グラデーションエクスポート

インラインエディタで作成したカスタムグラデーションを外部ファイルとして書き出すことができます。

| 形式 | 説明 |
|---|---|
| **GRD** | Photoshop レガシー v3 フォーマットで書き出します。他のツールや共同作業者とグラデーションプリセットを共有できます。 |
| **PNG** | 256×1 ピクセルの Premultiplied Alpha PNG として書き出します。他のエフェクトやアプリケーションでルックアップテーブルとして利用可能です。 |

### 8. お気に入り管理

よく使うグラデーションファイルを「お気に入り」として登録できます。お気に入りに登録されたファイルは `GradientMapSettings`（YMM4 の設定機構 `SettingsBase`）を通じてセッション間で保存され、ファイルセレクターのリスト上部に常に表示されます。

### 9. 多言語対応

ローカライズファイルを内蔵しており、YMM4 の言語設定に合わせて UI やパラメータ名、27 種類の合成モード名・説明文がすべて自動で切り替わります。
対応言語：日本語 (`ja-jp`)、英語 (`en-us`)、中国語簡体字 (`zh-cn`)、中国語繁体字 (`zh-tw`)、韓国語 (`ko-kr`)、スペイン語 (`es-es`)、アラビア語 (`ar-sa`)、インドネシア語 (`id-id`)

## パラメータ一覧

プロパティエディタで設定可能な項目は以下の通りです。

| パラメータ名 | 型 | デフォルト | アニメーション | 説明 |
|---|---|---|---|---|
| **グラデーション** | String (JSON) | (空) | ✗ | インラインエディタでグラデーションを直接定義します。値が存在する場合、グラデーションファイルより優先されます。 |
| **ファイル** | String | (空) | ✗ | グラデーションファイルのパスを選択します（サムネイル付きカスタムUI）。`.grd`, `.png`, `.jpg`, `.jpeg`, `.bmp`, `.gif`, `.tiff` に対応。 |
| **選択** | Int | 0 | ✗ | `.grd` ファイル内に複数のグラデーションが含まれる場合、インデックスで選択します（サムネイル付きドロップダウンUI）。 |
| **不透明度** | Animation | 100 % | ✓ | グラデーションマップの適用強度を `0〜100%` の範囲で設定します。0% で元映像がそのまま出力されます。 |
| **合成モード** | Enum | 通常 | ✗ | 27 種類の合成モードから選択します。 |
| **水平サンプリング** | Bool | ON | ✗ | グラデーションテクスチャのサンプリング方向を切り替えます。ON で水平（左→右）、OFF で垂直（下→上）方向にサンプリングします。 |

## 対応ファイル形式

| 形式 | 拡張子 | 説明 |
|---|---|---|
| **Photoshop グラデーション** | `.grd` | v3/v4 レガシーおよび v5 ディスクリプタ形式。RGB/HSB/CMYK/Lab/グレースケールのカラーモデルに対応。マルチグラデーション対応。 |
| **画像ファイル** | `.png`, `.jpg`, `.jpeg`, `.bmp`, `.gif`, `.tiff` | 画像の 1 行目（水平モード時）または 1 列目（垂直モード時）をグラデーションとしてサンプリングします。BGRA32 (Premultiplied Alpha) に変換して使用。 |

## 合成モード一覧

すべての合成モードは HLSL ピクセルシェーダー内で演算されます。

| # | モード名 | 説明 |
|---|---|---|
| 0 | **通常 (Normal)** | グラデーションマップの色をそのまま適用 |
| 1 | **輝度 (Luminosity)** | 元の色相と彩度を維持し、グラデーションの輝度を適用 |
| 2 | **色相 (Hue)** | グラデーションの色相方向に元の輝度を適用 |
| 3 | **比較(暗) (Darken)** | 各チャンネルの暗い方の値を選択 |
| 4 | **乗算 (Multiply)** | 元の色とグラデーションの色を乗算 |
| 5 | **焼き込みカラー (Color Burn)** | コントラストを上げて暗くする |
| 6 | **焼き込みカラー(リニア) (Linear Burn)** | 元の色とグラデーションの色を加算し 1 を減算 |
| 7 | **減算 (Subtract)** | グラデーションの色を元の色から減算 |
| 8 | **比較(明) (Lighten)** | 各チャンネルの明るい方の値を選択 |
| 9 | **スクリーン (Screen)** | 両色を反転して乗算し再反転 |
| 10 | **覆い焼きカラー (Color Dodge)** | コントラストを下げて明るくする |
| 11 | **覆い焼きカラー(加算) (Linear Dodge)** | 元の色とグラデーションの色を加算してクランプ |
| 12 | **加算(発光) (Add Glow)** | 元の色とグラデーションの色を加算し 1 でクランプ |
| 13 | **オーバーレイ (Overlay)** | 暗い部分に乗算、明るい部分にスクリーンを適用 |
| 14 | **ソフトライト (Soft Light)** | オーバーレイに似た、より柔らかいコントラスト |
| 15 | **ハードライト (Hard Light)** | グラデーションを基準にオーバーレイを適用 |
| 16 | **差の絶対値 (Difference)** | 元の色とグラデーションの差の絶対値 |
| 17 | **ビビッドライト (Vivid Light)** | グラデーション値に応じて焼き込みまたは覆い焼き |
| 18 | **リニアライト (Linear Light)** | グラデーション値に応じてリニア焼き込みまたは覆い焼き |
| 19 | **ピンライト (Pin Light)** | グラデーション値に基づいて明暗を置き換え |
| 20 | **ハードミックス (Hard Mix)** | 各チャンネルの合計に基づいて白または黒 |
| 21 | **除外 (Exclusion)** | 差の絶対値に似た、低コントラスト版 |
| 22 | **カラー比較(暗) (Darker Color)** | 輝度の低い方の色全体を選択 |
| 23 | **カラー比較(明) (Lighter Color)** | 輝度の高い方の色全体を選択 |
| 24 | **除算 (Divide)** | 元の色をグラデーションの色で除算 |
| 25 | **彩度 (Saturation)** | 元の色相と輝度を維持し、グラデーションの彩度を適用 |
| 26 | **カラー (Color)** | 元の輝度を維持し、グラデーションの色相と彩度を適用 |

## 使い方ガイド

### グラデーションマップを適用する
1. タイムラインで映像アイテム（動画・画像など）を選択します。
2. 映像エフェクトの追加から「グラデーションマップ」を選択します。
3. プロパティパネルにグラデーションマップの設定項目が表示されます。

### グラデーションファイルを使用する
1. 「ファイル」プロパティの「参照」ボタンを押してファイルダイアログを開きます。
2. `.grd` ファイルまたはグラデーション画像（PNG / JPG など）を選択します。
3. ドロップダウンリストには、選択したファイルと同じフォルダにある対応ファイルが自動的にリストアップされ、サムネイルで内容を確認しながらすばやく切り替えられます。
4. `.grd` ファイルに複数のグラデーションが含まれている場合、「選択」ドロップダウンが自動表示されます。名前とサムネイルを確認して使いたいグラデーションを選択してください。

### インラインエディタでグラデーションを作成する
1. 「グラデーション」欄にインラインエディタが表示されています。
2. グラデーションバー上をクリックして、新しいカラーストップを追加します。
3. カラーストップマーカーをドラッグして位置を調整します。
4. カラーピッカーで色を変更すると、リアルタイムにプレビューに反映されます。
5. 不要なカラーストップは右クリックメニューから削除できます（最低 2 ストップは必要です）。
6. エディタの内容は自動的に JSON としてプロジェクトに保存されます。

### グラデーションをエクスポートする
1. インラインエディタでグラデーションを作成・調整します。
2. 「GRD として書き出す」または「PNG として書き出す」ボタンをクリックします。
3. 保存先を指定してエクスポートします。GRD 形式は Photoshop やその他の互換ツールで読み込めます。

### 合成モードを使いこなす
- **通常**: 元映像を完全にグラデーションの色で置き換えます。色変換やモノクロ→カラー化に最適です。
- **ソフトライト / オーバーレイ**: 元映像のディテールを残しつつ色味を加える、自然なカラーグレーディングに適しています。
- **乗算**: 暗い方向に色を強調し、ヴィンテージやセピア調の表現に効果的です。
- **スクリーン**: 明るい方向に色を強調し、ハイキー表現やグロー効果の補助に使えます。
- **不透明度の併用**: いずれの合成モードでも「不透明度」パラメータで効果の強さを `0〜100%` で無段階に調整できます。

### お気に入りを活用する
1. ファイルセレクターのドロップダウンで、ファイルエントリのお気に入りアイコンをクリックします。
2. お気に入りに登録されたファイルはリスト先頭に常に表示され、異なるフォルダのファイルもすぐにアクセスできます。
3. お気に入り情報は YMM4 のプラグイン設定に保存され、セッションをまたいで保持されます。

## 注意事項

- グラデーションマップエフェクトは GPU を使用して描画します。ピクセルシェーダー 5.0 に対応していない極端に古い GPU 環境では、エフェクトが無効化される場合があります。
- `.grd` ファイルの「ノイズグラデーション」（`GrdT != 0`）はサポートしていません。ソリッドグラデーション（`GrdT == 0`）のみ読み込み可能です。
- ファイルセレクターのドロップダウンリストには、選択中のファイルと同じディレクトリにある対応拡張子のファイルが自動検索されて表示されます。ディレクトリ内に大量のファイルが存在する場合、サムネイル生成処理がバックグラウンドで行われます。
